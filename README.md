Структура закодированного файла

Файл формата .huf состоит из трех частей.

1. Размер исходного файла
Содержит количество байтов в исходном файле.
Тип: int.
Размер: 4 байта.

2. Таблица частот символов
Содержит частоты появления каждого возможного байта.
256 целых чисел типа int.
Размер: 256 × 4 = 1024 байта.
По данной таблице при декодировании восстанавливается дерево Хаффмана.

3. Закодированные данные
Содержит последовательность битов, сформированную с использованием кодов Хаффмана.
Биты упаковываются в байты и записываются в файл после служебной информации.

Особенности реализации.

Программа корректно обрабатывает:
пустые файлы;
файлы с одним уникальным символом;
текстовые файлы;
бинарные файлы.

Корректность работы подтверждена тестированием на текстовых и бинарных данных.
